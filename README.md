# FancyAI Skills Collection

**Premium AI insights and creative tools for creators, marketers, and retailers.**

FancyAI Skills are expert-level workflows designed for **all industries** — fashion, FMCG, beauty, tech, sportswear, luxury, and beyond. Each skill combines real business case studies with behind-the-scenes creative knowledge, giving AI assistants the ability to execute professional campaigns, direct photoshoots, generate product assets, and guide visual storytelling with the nuance of experienced creative directors.

Whether you're a brand marketer, a creative director, an e-commerce retailer, or a social media manager, FancyAI Skills help you move faster without sacrificing quality.

## About This Repository

This repository contains professionally crafted skills for **Marketing, E-Commerce, Fashion & Editorial, Digital Marketing, and Design** — all backed by real-world case studies and production knowledge.

Each skill is self-contained in its own folder with a `SKILL.md` file containing detailed instructions, prompt templates, and workflow guidelines.

All skills in this repository are open source under the **Apache 2.0 License** — free to use, modify, and distribute.

## Contributing

We welcome contributions! To add a skill:

1. Fork this repository
2. Create a branch (`git checkout -b skill/your-skill-name`)
3. Add your skill in its own folder with a `SKILL.md` file
4. Commit (`git commit -m 'Add your-skill-name'`)
5. Push (`git push origin skill/your-skill-name`)
6. Open a Pull Request

Please ensure your skill follows the standard structure and includes clear documentation.

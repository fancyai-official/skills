# Scene Inspiration Library (Load On Demand)

> Only read this file when selecting scenes for a project. The scene list is for inspiration — **always expand creatively, never repeat mechanically**.

## Core Principles
- **Avoid habitual scenes**: Don't always default to "café entrance", "gym", or "park running track"
- **Self-check before each generation**: Have I used this scene recently? What scene best suits this particular shoe?
- **Be specific with scene descriptions**: Don't just write "street" — write "modern art district brick wall backdrop, side natural light"
- **Scene quality requirement**: Beautiful, premium, design-forward — meets commercial advertising campaign standards

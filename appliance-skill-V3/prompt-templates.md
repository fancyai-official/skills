# 14-Style Complete Prompt Template Library

> Each style provides: **Six-Layer Structure Template** (with placeholders) + **Fill Instructions** + **Enhancement Keywords**
>
> Placeholder Guide:
> - `{product_name}` — Precise English product name (e.g.: wireless earbuds, hair dryer, robot vacuum)
> - `{main_color}` — Product main color (e.g.: matte black, pearl white, champagne gold)
> - `{material}` — Primary material (e.g.: matte plastic, brushed aluminum, glossy ABS)
> - `{brand_color}` — Brand color (e.g.: deep purple, forest green, coral red)
> - `{accent_color}` — Accent / glow color (e.g.: cyan blue, warm amber, electric green)
